# Datawrapper API

View live: <https://observablehq.observablehq.cloud/framework-example-datawrapper-api/>

This Observable Framework example demonstrates how to write a data loader in Python that generates [Datawrapper](https://www.datawrapper.de/) chart you can embed in your page.

The data loader lives in [`src/data/chart.txt.py`](./src/data/chart.txt.py).
